Installatie instructies OnTip hussel standalone
===============================================

1. EasyPHP Installer starten
2. Start EasyPHP server
3. Ga naar configuratie Appache en pas de poort aan als Easy PHP niet goed start (is standaard 80)
4. Pas in create_all_ tables.txt de naam van de vereniging aan
5. Kopieer de bestanden van de USB van folder Onip hussel naar c:\Program Files(x86)\EasyPHP.....\data\localweb\projects\OnTip Hussel
6. In TaakBalk rechtermuisknop op EasyPHP \Administration\
7. Open MySQL Administration 
8. Create Database ontip_hussel
9. Kopieer de inhoud van create_all_tables.txt en plak dit in het  commando van phpMyAdmin
10.In TaakBalk rechtermuisknop op EasyPHP \Localweb
11.Zoek de folder Ontip_hussel onder projects. Direct start de hussel op
12. Kopier de url in de browser
13. Maak dit de startpagina in je browser
